# 🥷 Schneider Ninja

A 2D endless runner game built with **Unity & C#**, developed during a co-op internship at **Schneider Electric**. The game was showcased live at **Innovation Summit Riyadh 2025**, where it attracted hundreds of visitors at the interactive gaming booth.

---

## 🎮 Gameplay

Guide the Schneider Ninja through obstacles, collect coins, and survive as long as possible. The game features multiple phases that increase in difficulty, keeping players engaged and challenged.

- **Avoid obstacles** to stay alive
- **Collect coins** to boost your score
- **Survive longer** to unlock harder phases

---

## ✨ Features

- 🎯 Endless runner mechanics with increasing difficulty
- 🪙 Coin collection system with score tracking
- 🌊 Multiple game phases with dynamic transitions
- 💥 Animations, sound effects, and game over screen
- ⏱️ 60-second timed gameplay sessions
- 🏆 Reward system based on performance

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/ReemNfis/schneider-ninja.git
   ```
2. Open the project in **Unity Hub**
3. Select the correct Unity version
4. Press ▶️ **Play** to start the game
